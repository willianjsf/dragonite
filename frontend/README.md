# Dragonite Client

Dragonite Client for a Federated Chat System. Using Svelte and Tailwind to create a webapp

## Pre-requisites

- Node.js 25.8+
- NPM 0.11+

## Install

1. Clone this repository
2. Access the client folder

```sh
cd client/dragonite
```

3. Instal all dependecies

```sh
npm install
```

## Usage

1. Run in developer mode

```sh
npm run dev
```

2. Run in production

```sh
npm run build
```

And look at `public` folder to see the compiled project.

## Technologies

- SvelteKit
- Tailwind CSS
- Matrix js SDK
